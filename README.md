# 📊 Google Play Store Rating Prediction

A machine learning-based web application that predicts app ratings using various features such as category, size, and genres. This project demonstrates data preprocessing, feature engineering, and regression modeling.

---

## 🚀 Features

* Predict app ratings using regression models
* Data preprocessing and handling missing values
* Feature engineering for improved model performance
* Interactive frontend for user input
* Efficient model training with optimized performance

---

## 🛠️ Tech Stack

* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Frontend:** HTML, CSS, JavaScript, Bootstrap

---

## ⚙️ How It Works

1. Dataset is collected and cleaned
2. Missing values are handled using mean/median techniques
3. Features are selected and transformed
4. Regression model is trained
5. User inputs app details
6. Model predicts the rating

## 💻 How to Run

1. Clone the repository:

```
git clone https://github.com/manjumedhasree/Google-Play-Store-Rating-Prediction-using-ML.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the application:

```
python app.py
```

4. Open browser:

```
http://localhost:5000
```
